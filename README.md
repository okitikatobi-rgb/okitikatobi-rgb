Heart Disease Prediction Using Machine Learning

Code: [](https://github.com/okitikatobi-rgb/Data-Analyst-Repository/blob/main/healthcare-1.ipynb)

Goal: This project aims to predict the likelihood of heart attack risk, enabling early detection, better diagnosis support, and improved patient outcomes in healthcare settings.

Description: In this project, the dataset contains patient information such as age, sex, chest pain type (cp), resting blood pressure (trestbps), cholesterol (chol), maximum heart rate (thalach), thalassemia (thal), and other clinical indicators. The data cleaning process involves checking for missing values, duplicates and fixing them. This is followed by performing exploratory data analysis (EDA) and the machine learning process using a Logistic Regression model to predict target variables. The target variable 1 indicates heart disease present (or high risk of heart attack) and 0 indicates no heart disease.

Skills: Data Handling & Preprocessing, Exploratory Data Analysis (EDA), Data Visualisation, Machine Learning, Healthcare Domain Knowledge, Tool Integration, Project Workflow, and Problem-Solving.

Technology: Python, Pandas, NumPy, Seaborn, Scikit-learn.

Result: Our findings indicate that there is a strong predictive signal describing a strong assertion that machine learning can be used for early detection of cardiovascular-related risks.
